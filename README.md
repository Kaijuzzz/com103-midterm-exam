How to Use the Program
Enter your details
Type your name
Enter your weekly budget
View categories
The program will show 5 expense categories with examples.
Record expenses (up to 4)
Choose a category number (1–5)
Enter 0 to skip
Type a description of the expense
Enter the amount spent
Automatic alert

If an expense is more than 25% of your budget, it will show:

! High Expense Alert!
View results
The program will display:
Your name
Weekly budget
List of expenses
Total spent
Remaining money
Budget status
Check your status
“Good at Budgeting.” → You are within budget
“Overspent! Reduce spending.” → You exceeded your budget
⚠️ Reminder
Enter numbers only for budget and amount.
Choose only 1–5 for categories.
